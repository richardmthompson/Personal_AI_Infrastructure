## Quick Start Workflow

### First Time (30 minutes)
1. Read [STRATEGY-AND-PLANNING-GUIDE.md](STRATEGY-AND-PLANNING-GUIDE.md) Section 1
   - Understand the 6-doc system
   - Follow "Your First Week" guide
2. Review [CONTENT-STRATEGY.md](CONTENT-STRATEGY.md) - Your 5 pillars
3. Check [NOV-2024-PLAN.md](NOV-2024-PLAN.md) - This month's theme
4. Open [WEEKLY-TRACKER.md](WEEKLY-TRACKER.md) - This week's tasks

### Each Month (1-2 hours, first Monday)
1. Follow [STRATEGY-AND-PLANNING-GUIDE.md](STRATEGY-AND-PLANNING-GUIDE.md) Section 3
2. Review [CONTENT-STRATEGY.md](CONTENT-STRATEGY.md) - Check pillars, add assets
3. Create new [MONTH]-PLAN.md using template
4. Update [WEEKLY-TRACKER.md](WEEKLY-TRACKER.md) for Week 1

### Each Week (1 hour, Monday morning)
1. Follow [STRATEGY-AND-PLANNING-GUIDE.md](STRATEGY-AND-PLANNING-GUIDE.md) Section 2
2. Review [WORKFLOW.md](WORKFLOW.md) - Batching schedule
3. Update [WEEKLY-TRACKER.md](WEEKLY-TRACKER.md) - This week's tasks
4. Check [NOV-2024-PLAN.md](NOV-2024-PLAN.md) - Month context

### Daily (10 minutes)
- Check [WEEKLY-TRACKER.md](WEEKLY-TRACKER.md) for today's tasks
- Engagement: Reply to comments, engage with 5-10 creators

### Quarterly (2-3 hours)
1. Follow [STRATEGY-AND-PLANNING-GUIDE.md](STRATEGY-AND-PLANNING-GUIDE.md) Section 4
2. Deep review of [CONTENT-STRATEGY.md](CONTENT-STRATEGY.md) - Adjust pillars if needed
3. Consult [IMPLEMENTATION-GUIDE.md](IMPLEMENTATION-GUIDE.md) if needed
